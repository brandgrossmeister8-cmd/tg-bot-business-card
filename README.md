# Telegram Бот-Визитка для Эксперта по Маркетингу

Простой и элегантный Telegram-бот для представления услуг эксперта по стратегическому маркетингу и системному продвижению.

## Возможности

- Приветствие и навигация через команду `/start`
- Информация об эксперте `/about`
- Список услуг и программ `/programs`
- Использование эмодзи и Markdown для красивого оформления
- Обработка неизвестных команд

## Установка и настройка

### 1. Клонируйте репозиторий или скачайте файлы

### 2. Установите зависимости

```bash
npm install
```

### 3. Создайте бота в Telegram

1. Найдите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный токен

### 4. Настройте переменные окружения

Создайте файл `.env` в корне проекта:

```bash
cp .env.example .env
```

Откройте `.env` и вставьте ваш токен:

```
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
PHOTO_URL=https://example.com/your-photo.jpg
```

**Настройка фото эксперта:**

Есть два способа добавить фото:

**Способ 1: Использовать URL фотографии**
- Добавьте в `.env` переменную `PHOTO_URL` со ссылкой на фото

**Способ 2: Использовать локальный файл**
- Поместите фото в папку `images/` (например, `images/photo.jpg`)
- В файле [bot.js](bot.js:39) раскомментируйте строку с локальным путем и закомментируйте строку с URL

### 5. Запустите бота

```bash
npm start
```

Для разработки с автоматической перезагрузкой:

```bash
npm run dev
```

## Команды бота

- `/start` - Приветствие и главное меню
- `/about` - Информация об эксперте
- `/programs` - Список услуг и программ
- `/quiz` - Интерактивный тест для подбора программы

